a.  Sudut 30° dan 60°


Segitiga ABC diatas merupakan segitiga sama sisi dengan AB = BC = AC = 2x cm dan ∠A = ∠B = ∠C = 60º. Dikarenakan CD tegak lurus AB, maka CD merupakan garis tinggi sekaligus garis bagi ∠C, sehingga ∠ACD = ∠BCD =30º. Dan diketahui  ∠ ADC = ∠ BDC = 90º. Titik D merupakan titik tengah AB, dimana panjang AB = 2x cm sehingga panjang BD = x cm.
Perhatikanlah segitiga CBD.
Kita gunakan teorema pythagoras maka diperoleh
CD² = BC² – BD²
CD  = √(BC² – BD²)
CD  = √((2x)²−x²)
CD  = √(4x²−x²)
CD  = √(3x²)
CD  = x√(3)
Dengan demikian diperoleh perbandingan sebagai berikut :
BD : CD : BC = x : x√(3) : 2x
BD : CD : BC = 1 :  √(3) : 2
