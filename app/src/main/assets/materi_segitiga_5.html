Segitiga ABC diatas adalah segitiga siku-siku sama kaki, dengan sudut B adalah sudut siku-siku dimana panjang AB = BC = x cm dan ∠ A = ∠ C =450
Kita gunakan teorema pythagoras maka diperoleh :
AC² = AB² + BC²
AC  = √(AB²+BC²)
AC  = √(x²+x²)
AC  = √(2x²)
AC  = x√2
Dengan demikian, diperoleh perbandingan sebagai berikut :
AB : BC : AC = x : x : x√2
AB : BC : AC = 1 : 1 : √2
